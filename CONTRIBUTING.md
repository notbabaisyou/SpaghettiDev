# Contribution Guidelines

1. AI generated contributions are permitted under the following conditions.

> You take full responsbility for the code you submit.

> You get what you give, provide garbage and expect garbage in response.

> The use of AI in any form must be disclosed.

> Your code must follow the code style of the project.

> Write-ups for bugs and PRs must be done by a human, you may use AI to fix up grammar and spelling.

2. Commit messages should be concise and get to the point.

3. Changes should be per area/per file, an exception is made
for very large changesets, where squashing everything into a single commit is allowed.

4. Bug fixes affecting a stable release and master should be submitted as different PRs, per branch.
